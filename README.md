# EBAC-Analista-Dados
Repositório criado para fins de estudo do curso de analista de dados da escola EBAC - (tudo é criado a fins de testes e estudos).

# Calculadora básica
A pasta **atividade_1** possui o desafio do módulo de lógica de programação com python. 
Nela você vai encontarr um programa simples que funciona em dois laços de repetição. Um verifica se as opções digitadas estão validas e o outro re executa o programa.
Após passar pelo laço de verificação ele solicita os numeros e faz uma das operações desejadas (soma, subtração, multiplicação e divisao).
Alem do código foi gerado um programa executavel para testar a aplicação sem necessidade de compilar o código.

# Comandos Linux
Na pasta **ComandosArquivoSH** a gente vai encontrar um arquivo de comandos onde indica como foi feia a criação de um script chamado calculadora.sh
Descrilçao dos comandos por linha:

1 touch: cria um arquivo 
2 nano: abre o editor para o arquivo que criamos
3 chmod 744: altera as permissões do arquivo
4 ls -l: lista os arquivos no diretório atual
5 ./: executa o arquivo com o nome indicado.

(A atividade do módulo de linux nçao informava inserções pra executar o script então foi informado apenas uma mensagem informativa com o comando echo, que permite exibir mensagens no console)
